Ricing so to learn the internals of the operating system tools

The set of tools are as follows:
- window manager: i3
- compositor: picom
- status bar: polybar
- background daemon: feh
- app launcher: rofi
- audio library: pipewire
- terminal emulator: kitty
- editor: neovim
- terminal tiling app: tmux
